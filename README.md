# 🤖 CodeAlpha FAQ Chatbot

An AI-powered FAQ chatbot built as part of the CodeAlpha Artificial Intelligence Internship. This chatbot is designed to answer common questions related to the CodeAlpha internship program using intelligent response matching and a user-friendly interface.

---

## 🚀 Features

- 💬 Smart FAQ-based chatbot
- 🧠 Semantic question understanding (handles different question formats)
- ⚡ Fast and accurate responses
- 🎨 Modern and responsive UI design
- 🕒 Chat history with timestamps
- 📋 Copy-to-clipboard functionality
- 🔄 Clear chat option
- 💡 Suggested questions for better interaction
- 😊 Friendly conversational responses with emojis

---

## 📚 Knowledge Coverage

The chatbot can answer 20+ questions related to:
- Internship details
- Duration (1 month / 3 months)
- Tasks requirement (2–3 tasks mandatory)
- Certification fee (₹99)
- LinkedIn requirements (mandatory updates)
- GitHub submission process
- Domains (30+ including AI, Web Dev, Data Science, etc.)
- Eligibility, perks, and project types

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Google AI (Gemini Model via AI Studio)

---

## 🎯 Project Objective

To build an intelligent chatbot that simulates a real-world customer support system by:
- Understanding user queries
- Matching them with relevant answers
- Providing a smooth and engaging user experience
